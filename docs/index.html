Features

制限のない翻訳体験

クラウドの制限から解放された、完全にローカルで動作する翻訳アプリ

完全無制限

文字数制限なし、API制限なし、月額料金なし。好きなだけ翻訳できます。

グローバルホットキー

どのアプリからでもショートカットキー一発で選択テキストを即座に翻訳。

システムトレイ常駐

バックグラウンドで静かに待機。必要な時にすぐに呼び出せます。

プライバシー保護

翻訳データは外部サーバーに送信されません。機密文書も安心して翻訳可能。

How it works

かんたん3ステップ

セットアップは数分で完了

1

ローカルLLMを準備

OllamaまたはLM Studioをインストールして、お好みの翻訳モデルをダウンロード。

ollama pull llama3
2

Local Translatorをインストール

Windows (.msi) または macOS (.dmg) のインストーラーをダウンロードして実行するだけ。

3

翻訳開始

テキストを選択して Ctrl+Alt+L を押すだけ。どのアプリからでも即座に翻訳。

Ctrl + Alt + L
